# 🌐 Edge Computing Lab

This repository contains all experiments performed in the Edge Computing Lab using Raspberry Pi 5, Docker, and Python.

---

# 📚 Experiments Overview

## 🔹 Experiment 1(A)
### Temperature & Humidity Monitoring using DHT Sensor

- Interfaced DHT11/DHT22 sensor with Raspberry Pi 5
- Used GPIO (RP1 architecture) and Python
- Displayed real-time temperature and humidity

📁 Folder: `Exp1A_DHT`  
📸 Output: See `output.png`

---

## 🔹 Experiment 1(B)
### Virtual Weather Station (Flask + API)

- Built a web dashboard using Flask
- Fetched real-time weather data using Open-Meteo API
- Displayed temperature and humidity on browser

📁 Folder: `Exp1B_Weather`  
📸 Output: See `output.png`

---

## 🔹 Experiment 3
### Docker Microservices (REST API Communication)

- Created two services:
  - Data Logger (FastAPI)
  - Sensor Simulator
- Implemented communication using REST API
- Logged temperature data continuously

📁 Folder: `Exp3_Docker`  
📸 Output: See `output.png`

---

## 🔹 Experiment 4
### DoS Attack Simulation & Container Isolation

- Simulated CPU exhaustion using multiprocessing
- Demonstrated Denial of Service (DoS)
- Highlighted need for resource control in containers

📁 Folder: `Exp4_DoS`
